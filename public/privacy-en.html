12. GitHub API Integration

Our website displays recent GitHub repositories via the GitHub REST API.

12.1 How It Works

  • Your browser fetches public repository data from GitHub
  • No personal data about you is sent to GitHub — only standard HTTP logging
  • Data is processed entirely in your browser (client-side)

12.2 Data Shared with GitHub

  • Shared: Public repository metadata (name, description, stars, etc.)
  • Not shared: Your personal data, behavior, or identity

12.3 GitHub as Data Processor

  • Processor: GitHub Inc., 88 Colin P. Kelly Jr. St, San Francisco, CA 94107, USA
  • Legal basis: Art. 6(1)(f) GDPR (legitimate interest: portfolio transparency)
  • Privacy: See GitHub Privacy Statement

12.4 Caching

Repository data is cached in your browser for 24 hours to reduce API calls.

12.5 Opt-Out

You can block requests to api.github.com using browser extensions.
